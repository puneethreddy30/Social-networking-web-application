************************************************************************************************************************************************************
Features Implemented in the Project 
************************************************************************************************************************************************************
1.	MySQL- Implemented in Registration.java, Login.java, Validate.java, Products.java, Likes.java and Rent Items.java pages.

2.	SaxParser- Implemented in Products.java.

3.	MongoDB- Implemented in ViewReview.java and Submit.java.

4.	.XML- Implemented the .XML File to load the products into HashMap using SaxParser.

5.	HashMaps- Implemented to load Products in HashMaps in the file Products.java from Products.XML.

6.	JSP Pages- Implemented for Search, Logout, Play Activity and  Post Activity.

7.	SaxParser- It used to load the Products from .XML file to HashMap in Products.java.

8.	Ajax Auto Completion- Ajax is implemented for searching the activities posted by the users.

9.	Post Activity- A user can post an Activity  that he is interested in or is going to perform at his selected location.

10.	Search Activity- A user can search for an Activity using the Search Activity box based on the interest.

11.	Join Activity- Any user can send a message to the users who have posted the activity and would like to join the activity.

12.	Rent Products- Products can be rented from Meet-A-Mate.

13.	Product Review- A user can provide a review based on his experience of the products use.

14.	Product Like- A product  can be liked in the products Page. 

15.	Read Review- Users can read the reviews provided by the users who have already used the products.

16.	Profile- Every User has a profile in which he or she can show there personal information, other users have the option to message the users in there profile page.

17.	Twitter – Tweets from Twiiter are being fetched for Events happening in and around the Chicago.
